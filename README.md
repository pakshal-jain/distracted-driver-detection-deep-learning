# 🚗 Distracted Driver Detection

Deep learning-based system to detect driver distraction using computer vision.

## 📸 Demo
![Demo](assets/driver.gif)

## 🚀 Features
- Detects distracted driving behavior
- CNN-based image classification
- Real-time applicable model

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV

## 📂 Project Structure
- `distracted_driver_detection.ipynb` – model
- `assets/driver.gif` – demo output

## ▶️ How to Run
1. Open notebook in Jupyter
2. Install required libraries
3. Run all cells

## 🎯 Use Case
Used in smart vehicle systems for improving driver safety.
